This is a Small Prototype of a File Organizer.
It Organizes Files in the Current Directory into Folders based on their extension.

It uses a Dictionary to store the folders and their extensions.
It uses a Loop to iterate through the files in the current directory.
It uses a Conditional Statement to check the extension of each file.
It uses a Dictionary to store the summary of the files.
